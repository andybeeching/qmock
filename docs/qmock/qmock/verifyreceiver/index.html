    class method QMock.verifyReceiver 

    QMock.verifyReceiver( receiver [, raise] ) → Boolean | Exception
    • receiver (Mock): mock / receiver object to test
    • raise (Function) optional: Function to handle false comparison results

    Verifies the receiver object (the parent mock object) first, then individual members. Only passes if whole object tree passes, else throws exception (fail fast).
